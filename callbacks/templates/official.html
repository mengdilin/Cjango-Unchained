Simple and Fast C++ Web App Framework

Cjango (Cjango-Unchained) is a C++ web App Framework inspired by Python's Django. It provides fast response with interpreter-language-like usability by Dynamic Loading.

Chango was designed and implemented as a class project of Modern C++ Design class with Prof. Bjarne Stroustrup. It's written in C++14.

Team members

  • Mengdi Lin
  • Yi Qi
  • Yasutaka Tanaka

Features

  • Asyncronous HTTP request handling
  • Regular expression URL matching
  • Template file rendering
  • HTTP 1.0 GET/POST compatible
  • HTTP Session support
  • Dynamic Callback Loading
